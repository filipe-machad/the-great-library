# Guia de Estilo — The Great Library

## Tom Narrativo: Assombro Sereno

O tom canônico deste universo é o **Assombro Sereno**: místico, humano e contido.

- Tragédia sem grotesco. Horror sutil no limite do espiritual.
- Escrita **elegante e rebuscada**, porém acessível.
- Criatividade poética é bem-vinda em momentos oportunos: enigmas, descrições de azü, cenas de alta carga emocional.
- Referências: Tolkien, Martin, Rothfuss.

---

## O que Fazer

- Descrever azü com **sensações físicas reais**: frescor mineral, sabor metálico, escorrer tátil.
- Usar linguagem sensorial densa: texturas, temperaturas, sons, cheiros.
- Manter o mistério — sugerir mais do que explicar.
- Respeitar o silêncio como elemento narrativo.

## O que Não Fazer

- Tom épico exagerado ou linguagem de videogame.
- Magia como efeitos visuais espetaculares sem custo.
- Simplificar emoções complexas em rótulos.
- Emojis em conteúdo narrativo.

## Formatação de Texto

### Markdown

- Títulos: `#` para título do documento, `##` para seções, `###` para subseções.
- Separadores `---` entre seções principais.
- Tabelas para dados estruturados.
- Blockquotes `>` para citações in-world ou notas de design.
- Negrito para termos-chave na primeira aparição.
- Itálico para ênfase narrativa ou termos estrangeiros.

### Idioma

- Conteúdo em **pt-BR**.
- Termos próprios do cenário com grafia oficial (azü, Arin, Vigília-d'Alva).
- Nomes de arquivo em **kebab-case** sem acentos.
- Placeholders: `[nome a definir]` para elementos pendentes.
- Conflitos: `[CONTRADIÇÃO ENCONTRADA]` para inconsistências a resolver.
