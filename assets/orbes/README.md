# Referências Visuais — Orbes

Imagens de referência para a representação visual de **orbes** — núcleos de azü condensada.

Os orbes são pequenos, densos, de cores variadas, com padrões internos que lembram **nebulosas** e **universos em miniatura**. Cada um carrega uma assinatura cromática única que varia conforme a origem da azü e o processo de condensação.

> Créditos das imagens: **Ryan Smith** (conceitos de orbes/esferas mágicas)

## Uso Narrativo

Estas imagens servem como guia visual para:

- Descrever a aparência de orbes quando encontrados por personagens
- Representar a variação cromática da azü condensada
- Transmitir a ideia de "nebulosas internas" que caracterizam os orbes no lore
